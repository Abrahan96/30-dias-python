# Día 01 — Variables y tipos de datos

## Objetivo

Comprender qué es una variable, cómo almacena información y cuáles son los tipos de datos básicos de Python.

## Conceptos

- Variables
- `str`
- `int`
- `float`
- `bool`
- `type()`
- Convenciones para nombrar variables

## Ejercicio principal

Crear un programa que solicite el nombre, edad, ciudad y carrera del usuario, y luego muestre una presentación formateada.
